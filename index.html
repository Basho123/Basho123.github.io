Weather App

This is a plain and simple weather app that shows data for current weather, hourly, extended (3 hour interval) and weekly day by day.

  • Responsive for both mobile devices and desktop
  • Simple interface with most important data
  • Gets the user current location.

Tanks Game

This is a game that was developed from scratch, using p5.js canvas library elements. All the models, textures and audio used in the game are downloaded free. Game physics, collision formula and concept are original idea. Altough there are much more advanced game developing engines, this game is intentionally coded in Javascript for practicing purposes, and is mainly CPU based as it exists inside browser's engine. Enemy AI movements are randomized, and enemy AI targeting is devised so if you stand long enough they will acquire you as target, so get moving!

  • Not optimized for mobile phone or tablet (PC game).
  • High score and current score
  • Options consist of difficulty level and detail level to optimize for player
  • Hangar before each level. Upgrade system for the player tank, money reward system
  • 3 levels, dusk, day and dawn
  • Future implementation of multiplayer (through fetching posisions of created elements)

Login/Register Page

The page is in Macedonian language for practicing purposes. May be translated to english for demonstration purpose.

  • One page simple login/register form
  • Has validations for password and name length

Calculator

A simple calculator coded without using Javascript's eval() function. But this calculator watches for the advantage of multiplying and division before adding and subtracting.

  • Mathematically correct.
  • Gives true result when adding subtracting multiplying or dividing at once
  • Has buttons for power on/off and sound on/off
  • Can be used from keyboard numpad keys
  • Optimized for both PC and mobile phone

Morse Code Converter

Converter from morse code to text and vice versa. Can also decode special characters such as exlamation marks and question marks.

  • Simple interface and functionality

Farm Simulation

Farm Simulation starts from the moment the page is opened and workers come on the farm, tractors are bought, animals are bought, animals get sick, tourists are coming, veterinary regular check, random inspection check for worker parameters. Just play it and watch the simulation happening!

  • Artificial world with it's own logic enclosed within class bounds
  • Resources and assets menu to track the momentary status of the farm
  • Options to upgrade the farm property with more accomodation for animals and workers

Numbers to words converter app

As the headline states, this application converts numbers into words.

  • Optimized for both mobile and desktop use
  • Can convert up to 1015 numbers in to words
