Tanks Game

date created: 15.4.2021

This is a game that was developed from scratch, using p5.js canvas library elements. All the models, textures and audio used in the game are downloaded free. Game physics, collision formula and concept are original idea. Enemy AI movements are randomized, and enemy AI targeting is devised so if you stand long enough they will acquire you as target, so get moving!

    Controls:
  • WSAD - movement
  • NUMPAD 2468 - aiming
  • SPACE - Shoot
  • C - Camera

    Description:
  • Created with p5.js
  • Not using external physics library
  • Not optimized for mobile phone or tablet (PC game).
  • Options consist of difficulty level and detail level to optimize for player
  • Hangar before each level. Upgrade system for the player tank, money reward system
  • 3 levels, dusk, day and dawn
